# API Catálogo de Jogos Zerados

## Instruções de execução
1. Executar `npm install`
2. Executar `node index.js`
3. Acessar [http://localhost:3000/api-docs](http://localhost:3000/api-docs) para visualizar e testar os endpoints no Swagger UI.

## Estrutura do projeto
- index.js
- routes/games.js
- controllers/gamesController.js
- service/gamesService.js
- model/game.js
- middleware/auth.js
- data/games.json
- resources/swagger.json
